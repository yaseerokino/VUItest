Student VA Voice User Interface for Computer Science Students
Possible Commands
  • Greetings
  • -> You can say a greeting (e.g. "Hello", "Hi", "Howdy")
  • -> You can say thank you after a response
  • General Commands
  • -> Search for something (e.g. meaning of a word )
  • -> Ask for the Antonym or synonym of a word
  • -> Ask to learn a programming language. (e.g. Javascript, Python)
  • -> Ask How you can reference
  • -> Ask about a module. Note:. Say "course" or "courses" if the assistant can't recognise "module"
  • -> Ask how you can should write your dissertation


Please always click on the button before saying a command
